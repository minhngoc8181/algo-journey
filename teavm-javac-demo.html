TeaVM JavaC Demo

A real in-browser Java demo using teavm-javac: compile Java source in the browser, generate WebAssembly, run it, and report Accepted, Compile Error, or TLE.
This page uses the official teavm-javac browser pipeline. It compiles Solution.java together with a generated RunnerMain.java, then runs the generated WebAssembly in a Worker and terminates that Worker on timeout. Depending on browser support and network access to TeaVM assets, the first load may take a while.
