TeaVM JavaC Input Demo

A real in-browser Java demo using teavm-javac: compile Java source in the browser, generate WebAssembly, inject custom stdin, and run programs that read from a generated Input helper.
This page uses the official teavm-javac browser pipeline. It compiles Main.java together with a generated Input.java and RunnerMain.java. The runner injects the textarea contents into Input before calling Main.main(...). TeaVM assets are bundled locally in assets/teavm; open this page through a local HTTP server instead of file://.
